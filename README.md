# HIPPA-DGCN：Survival Analysis on Whole Slide Images via Hyper-Image Patches and Position-Aware Dense Graph Convolutional Networks
This repository contains the PyTorch implementation of the method proposed in the paper "HIPPA-DGCN：Survival Analysis on Whole Slide Images via Hyper-Image Patches and Position-Aware Dense Graph Convolutional Networks".

# Datasets Used
To evaluate the model's robustness and generalization, we utilize seven diverse datasets from TCGA and CPTAC, covering multiple cancer types and preparation methods:
TCGA Cohort : Comprises TCGA-BRCA, BLCA , GBMLGG , UCEC , and LUAD. These formalin-fixed paraffin-embedded slides are used for the primary experimental evaluations.
CPTAC Cohort : Includes CPTAC-LSCC  and LUAD . These frozen sections serve as external validation to test the model's cross-modality generalizability.
